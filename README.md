✈️ Flight Management System (C++ Data Structures Project)
📌 Description

A console-based Flight Management System built in C++ using advanced data structures including BST, AVL Tree, and Graph. The system manages flight records, compares search performance, and finds routes between airports using BFS.

⚙️ Features
Add, search, and delete flights using BST & AVL
Compare BST vs AVL search performance
Track AVL rotations for balancing
Airport route planning using BFS
Tree visualizers (BST & AVL)
File handling for persistent data storage
Input validation using regex
🧠 Concepts Used

BST, AVL Tree, Graphs, BFS, Recursion, File Handling, OOP, Regex Validation, Time Complexity Analysis
